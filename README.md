# Juego de Memoria Pokémon

Un pequeño juego de memoria construido con **React**, **Vite** y la [PokeAPI](https://pokeapi.co/), donde el objetivo es encontrar todos los pares de Pokémon ocultos en el tablero.

---

## Descripción
El juego consiste en un tablero de 16 cartas (8 pares de Pokémon). Las cartas están boca abajo y el jugador debe hacer clic en dos cartas para ver si son iguales. Si lo son, se quedan boca arriba; si no, se vuelven a ocultar. El juego termina cuando se encuentran todos los pares.

---

## Tecnologías Utilizadas
- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **Vite**: Herramienta de construcción rápida para proyectos modernos.
- **PokeAPI**: API pública para obtener información y sprites de Pokémon.

---

## Instalación

1. **Clona el repositorio** o descarga el código.
2. **Navega al directorio del proyecto**:
   cd tds-front
3. npm install
4. npm run dev
